# PeerMeet

PeerMeet is a real-time video conferencing application built with WebRTC, Socket.io, and React.

## Features

- **HD Video & Audio**: Crystal clear face-to-face communication.
- **Real-time Chat**: Text messaging support during calls.
- **Multi-participant**: Support for multiple peers in a room.
- **Instant Connect**: Low latency peer-to-peer connections.
- **Modern UI**: Clean, light, and subtle interface design.

## Tech Stack

- **Client**: React, Vite, CSS Modules (Subtle Light Theme)
- **Server**: Node.js, Express, Socket.io
- **Communication**: WebRTC (Simple-Peer)

## Getting Started

1.  **Install Client Dependencies**:
    ```bash
    cd client
    npm install
    ```

2.  **Install Server Dependencies**:
    ```bash
    cd server
    npm install
    ```

3.  **Run Development Server**:
    - Client: `npm run dev` (in `client/`)
    - Server: `node server.js` (in `server/`)
